How It Works

Three steps. No build tools. No config.

🔍

1. Scan

On DOMContentLoaded, chaistrap queries all elements whose class contains chai-.

⚙️

2. Parse

Each class name is matched against regex patterns. chai-p-4 → strip prefix → match spacing rule → compute 16px.

✨

3. Apply

Object.assign(el.style, styles) writes inline styles. The chai-* class is then removed from the DOM.
